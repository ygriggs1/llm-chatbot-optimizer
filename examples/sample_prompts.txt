# Sample Prompts for Prompt Engineering and RAG

## Base Prompts
1. **Question:** What are the main differences between supervised and unsupervised learning?
   **Answer:** Supervised learning uses labeled data to train algorithms, while unsupervised learning works with unlabeled data to find hidden patterns.

2. **Question:** Can you explain the concept of overfitting?
   **Answer:** Overfitting occurs when a model learns the training data too well, capturing noise and outliers rather than the actual underlying pattern.
