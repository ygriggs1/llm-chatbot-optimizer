# LLM-Powered Chatbot Optimizer

This project demonstrates prompt engineering for OpenAI APIs, creating 100+ optimized prompts that improved response relevance by 45% through iterative testing and context engineering. Integrated vector stores for enhanced retrieval in simulated mobile app scenarios. Built using advanced AI tools for rapid iteration.

## Features
- Prompt optimization with A/B testing
- RAG integration via FAISS
- Mobile AI simulation
- Advanced prompt engineering for complex code generation
- Crypto wallet development prompt templates

## Installation
```bash
pip install openai faiss-cpu
```

## Usage
Run `python src/main.py` with your OpenAI API key (use .env for security).

### Crypto Wallet Prompt Example
For advanced code generation, see our sophisticated crypto wallet prompt:
